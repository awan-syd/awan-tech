# 阿晚科技官网 - 表单激活完整教程

## 问题说明

FormSubmit.co 要求你提供一个**实际存在的表单页面**来激活邮箱。

---

## 激活步骤（按顺序操作）

### 第1步：打开你的联系页面

用浏览器打开：
```
D:\company_website_template\pages\contact.html
```

或者直接双击 `contact.html` 文件。

### 第2步：填写并提交测试表单

1. 填写任意测试信息（不需要真实）：
   - 姓名：测试
   - 电话：13800138000
   - 邮箱：158964224@qq.com（**必须是你自己的邮箱**）
   - 公司名称：测试
   - 需求描述：电商商城SaaS
   - 留言内容：测试留言

2. 点击"提交留言"

### 第3步：查收确认邮件

打开你的QQ邮箱（158964224@qq.com），查找来自 `noreply@formsubmit.co` 的邮件：

```
发件人: FormSubmit <noreply@formsubmit.co>
主题: Confirm email for 158964224@qq.com

邮件内容会包含一个 "Confirm" 或 "Activate" 按钮
```

**点击邮件中的确认按钮。**

### 第4步：激活成功

确认后，你会看到提示："Email confirmed! You can now receive emails."

---

## 如果收不到邮件

1. **检查垃圾邮件文件夹**
2. **搜索关键词**：`formsubmit`
3. **等待1-2分钟**：有时邮件会有延迟
4. **检查邮箱地址**：确保填写的是 `158964224@qq.com`

---

## 激活后测试

1. 再次打开 `contact.html`
2. 填写并提交表单
3. 这次你应该直接收到邮件了！
